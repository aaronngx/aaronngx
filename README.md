## üìå Notable Projects

### üß¨ Evaluating Data and Feature Efficiency of Variational Quantum Classifiers for Genomic Cancer Diagnostics
Python, Qiskit, Quantum Machine Learning, NumPy, pandas, Matplotlib | August 2025 ‚Äì December 2025

- Developed hybrid quantum-classical framework implementing QKSVM and VQC classifiers with 4 quantum feature maps including amplitude, angle-RY, ZZ entanglement, and BPS circuit, achieving 99.3% dimensionality reduction from 7,129 genes to 4-50 features using ANOVA F-test, SNR, and SCAD-SVM regularization
- Achieved 4√ó qubit efficiency through logarithmic amplitude encoding with 16 features encoded in 4 qubits compared to 16 qubits for linear encoding, with convergence at epoch 30, final loss of 0.42, and validation across 420 comprehensive experimental configurations
- Implemented rigorous validation framework with stratified k-fold CV, LOOCV, and isolated independent test set of 34 patients from 72 total following Golub et al. 1999 methodology to prevent data leakage and ensure robust generalization

### üí∞ YFO ‚Äì Your Finance Officer
Next.js, React, TypeScript, FastAPI, PostgreSQL, Redis, Celery, Ollama LLM, Qdrant, Docker | December 2025

- Built automated expense tracking platform processing 50+ Gmail receipts/minute using hybrid pattern-matching and local LLM fallback with Gemma 2B model, achieving sub-5s sync latency and 99%+ accuracy with comprehensive deduplication
- Designed star schema data warehouse with PostgreSQL and distributed Celery task queue, enabling real-time analytics dashboards, semantic search via Qdrant vector DB, and scheduled sync jobs with exponential backoff retry logic
- Deployed production-ready using Docker Compose with observability including structured logging, metrics, health checks, CI/CD pipelines, OAuth 2.0 authentication, and multi-user family budget support
